01

True Centre Logo

CSS Grid 1fr auto 1fr keeps the logo mathematically centred regardless of how wide the left or right columns grow.

02

Overlay Drawer

position: absolute; top: 100% inside the sticky header — the page never shifts when the mobile menu opens.

03

Theme Persistence

Light/dark preference is saved to localStorage. Both the desktop toggle and mobile toggle share one setTheme() function.

04

Clean Drawer

The mobile hamburger drawer shows only the brand logo and navigation links — no auth buttons, no redundant theme toggle cluttering the drawer.
