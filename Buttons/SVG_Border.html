5 Variants / SVG Border

SVG Border Buttons

Five ways to animate an SVG stroke around a button — from hover-triggered draws to continuous loops, marching dashes, and chasing dots.

A — Draw on Hover

stroke-dashoffset transitions from full perimeter → 0 when you hover. Pauses until the next hover.

B — Rotating Segment

A partial arc (35% of perimeter) continuously orbits the button using a linear infinite animation.

C — Marching Ants

Evenly-spaced dashes travel around the border in a continuous loop. Classic selection-style animation.

D — Draw → Hold → Erase Loop

The border fully draws itself, holds for a beat, then erases — on a seamless infinite loop with ease-in-out timing.

E — Chasing Dots

Tiny dot segments chase each other around the border, each offset by an equal fraction of the perimeter.

↑ Sections B – E animate continuously  ·  Section A animates on hover
